Koch Curve

Normal: koch_curve(t, 4, 20)

Modifications:

  • The colors for each koch curve will be random
  • Every curve is a random length less than the length parameter
  • There is an extra left turn 120 degrees and an extra koch_curve command which follows the same pattern as the other functions: a random color and a random length less than the length parameter
Modified: koch_curve(t, 6, 50) 

 
def koch_curve(t, depth, length):
    if depth == 1:
        t.fd(length)
    if depth > 1:
        koch_curve(t, depth - 1, random.randrange(length) )
        t.color(random.randrange(255),random.randrange(255),random.randrange(255))
        t.lt(60)
        koch_curve(t, depth - 1, random.randrange(length) )
        t.color(random.randrange(255),random.randrange(255),random.randrange(255))
        t.rt(120)
        koch_curve(t, depth - 1, random.randrange(length) )
        t.color(random.randrange(255),random.randrange(255),random.randrange(255))
        t.lt(60)
        koch_curve(t, depth - 1, random.randrange(length) )
        t.color(random.randrange(255),random.randrange(255),random.randrange(255))
        t.lt(120) 
        koch_curve(t, depth - 1, random.randrange(length) )

Sierpinkski Triangle

Normal: sierpinski(t, 5, 200)

Modifications:

  • Pensize is random between the ranges of 1 and 5
  • As the depth increases, the length increases by a multiplier of 1.5x
  • As the depth increase, the triangles will alternate between the colors red and blue
  • A sierpinski function has been removed and most of the turning commands have also been removed to avoid being incredibly messy
  • The fd command tells the turtle to move forward by length instead of length/2
Modified: sierpinski(t, 6, 10) 

   
def triangle(t, size):
    t.lt(60)
    t.fd(size)
    t.rt(120)
    t.fd(size)
    t.rt(120)
    t.fd(size)
    t.rt(180)

def sierpinski(t, depth, length):
    if depth == 1:
        triangle(t, length)
    if depth > 1:
        t.pencolor("red")
        t.pensize(random.randrange(5))
        sierpinski(t, depth - 1, length * 1.5 )
        t.penup()
        t.fd(length)
        t.pendown()
        t.rt(60)
        t.pencolor("blue")
        sierpinski(t, depth - 1, length * 1.5 )

Fractal Tree

Normal: tree(t, 4, 50, 20)

Modifications:

  • The command t.bk(size) in depth == 0 has been deleted
  • The "tree" is made up of many random colours
  • t.lt(2 * angle) has been changed to t.rt(2 * angle)
  • The other tree functions called in the modification now contain two times the original angle measure
  • At the end of the function t.bk(size) has been changed to t.bk(size * 3)
Modified: tree(t, 6, 10, 30) 


def tree(t, depth, size, angle):
    t.pensize(5)
    if depth == 0:
        t.color(random.randrange(255), random.randrange(255), random.randrange(255))
        t.fd(size)
    else:
        t.fd(size)
        t.rt(angle)
        tree_mod(t, depth-1, size, angle * 2)
        t.rt(2 * angle)
        tree_mod(t, depth-1, size, angle * 2)
        t.rt(angle)
        t.color(random.randrange(255), random.randrange(255), random.randrange(255))
        t.bk(size * 3)
